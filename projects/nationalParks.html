National Parks Ecology

Project Overview

This application provides an interactive platform to explore ecological metrics and park data across Canada. It integrates user authentication, dynamic data presentation, and a clean UI to facilitate environmental education and research.
Goal: A collaborative PHP-based web application that visualizes ecological data across Canada's national parks. This project was developed by Tashrif, Joyce, and Gabi to support environmental awareness and data-driven exploration


Programs and Technologies Used

{Html}{Css} {JavaScript} {PHP} {MySQL}


Feature

• User Authentication: Secure login system for personalized access.

• Data Visualization: Display of ecological and park data from CSV datasets.

• Responsive Design: Accessible on various devices with a user-friendly interface.

• Modular Codebase: Organized PHP structure for maintainability.
